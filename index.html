projects

covid-19 dashboard

This project(Covid-19 Economic Impact) analysis the impact of covid-19 on global economies, financial markets, exchange rates, oil prices, and weather conditions.

Using Excel for data cleaning, SQL for data extraction, Power BI for visualization, and DAX for advanced calculations, the dashboard provides insights into how the pandemic influenced key economic indicators.

The objectives and key insights include covid-19 Case Trends and Growth, Economic and Market Impact, External Factors and Correlations.

Heart Health Insight Nigeria

This project(Heart Health Insight Nigeria) analysis youth and adult heart condition in Nigeria,the category at which heart attack occur in respect to age category, Family history, Blood pressure impact, and Regional heart attact distribution.

Using Excel for data cleaning, Power BI for visualization, and DAX for advanced calculations, the dashboard provides insights into how various factors influenced Heart health in Nigeria.

Walmart Sales Project

Walmart Sales Project analyze various factors that influence sales trends using SQL Server Management Studio (SSMS) and power BI.

The dataset includes historical sales, markdown events, and economic indicators such as CPI, Unemployment, and Fuel Prices.
